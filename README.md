#### quick start

执行命令：`mvn tomcat7:run`

打包项目：`mvn clean package`

以上命令均可以使用参数来指定环境，如<br>
`mvn -Prelease clean package` （生产包）<br>
`mvn -Ptest tomcat7:run` （测试环境启动）
